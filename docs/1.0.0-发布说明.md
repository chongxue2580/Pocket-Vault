# Pocket Vault 1.0.0 发布说明

## 版本信息

- 产品名称：Pocket Vault / 拾光盒
- 版本号：`1.0.0`
- 平台：Android
- 仓库地址：<https://github.com/chongxue2580/Pocket-Vault>

## 版本定位

`1.0.0` 是首个可公开发布的正式版本，目标是交付一个可离线使用、可继续迭代、体验完整的本地收藏应用。

这个版本重点完成了以下闭环：

- 四类内容统一收纳
- 本地数据持久化
- 搜索 / 分类 / 详情 / 编辑闭环
- 加密空间与主密码能力
- 深色 / 浅色 / 简白主题
- 手机可直接安装使用

## 加密逻辑说明

### 普通加密

- 内容仍可出现在首页 / 分类 / 搜索中
- 进入详情页前需要输入主密码
- 可选择标题是否在列表中公开显示

### 超级加密

- 内容不出现在普通首页流中
- 通过首页下拉进入加密空间查看
- 加密空间需要主密码验证
- 若配置二次密码，则还需二次验证

### 删除保护

- 普通内容可直接删除
- 加密内容删除前需要再次输入主密码确认

## 设计与实现亮点

- 本地优先，不依赖账号系统
- 使用 Room 持久化
- 使用 Compose 构建 UI
- 使用 Hilt 管理依赖
- 敏感字段支持加密存储
- 统一的数据模型支持未来扩展

## 建议发布文案

### 简版

Pocket Vault 1.0.0 正式发布。  
这是一个本地优先、离线可用的 Android 收藏应用，支持链接、文字、图片、账号密码四类内容收纳，并提供主密码、二次密码与加密空间能力。

### 商店 / 仓库长版

Pocket Vault（拾光盒）是一款本地优先的 Android 收藏应用。  
你可以用它来保存网站链接、灵感文字、图片和账号密码，并通过收藏夹、标签、搜索、分类和加密空间进行统一管理。  
1.0.0 版本重点完成了内容管理、主题切换、加密能力和基础交互闭环，可作为首个正式公开版本发布。
